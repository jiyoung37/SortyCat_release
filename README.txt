SortyCat

SortyCat helps you rename and organize photos and videos by capture date.

Features
- Rename media files by capture date
- Organize files into date-based folders
- Works with mixed media from phones, cameras, and GoPro
- Drag and drop support
- Preview changes before renaming

How to use
1. Run SortyCat.exe
2. Drag and drop a folder, or click Browse
3. Click Preview
4. Review the proposed changes
5. Click Rename and Organize

Important
- Always test on a copy of your files first
- Back up important files before use
- If Windows shows a warning, click "More info" and then "Run anyway"
