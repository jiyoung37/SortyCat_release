# 🐱 SortyCat

SortyCat helps you automatically rename and organize your photos and videos by capture date.

---

## ✨ Features

* 📅 Rename media files by capture date
* 📂 Automatically organize files into date-based folders
* 📱 Works with files from Android, iPhone, GoPro, etc.
* 🖱 Drag & drop support (included in the executable)
* 🔍 Preview changes before applying

---

## 🚀 How to Use

1. Run `SortyCat.exe`
2. Drag and drop a folder, or click **Browse**
3. Click **Preview**
4. Review the changes
5. Click **Rename and Organize**
