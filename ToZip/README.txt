SortyCat helps you rename and organize media files by capture date.

✨ Features
Rename files using capture date
Organize files into folders automatically
Supports mixed sources (Android, iPhone, GoPro)
Drag and drop support included
Preview before applying changes

🚀 How to use
Run SortyCat.exe
Drag and drop a folder or click Browse
Click Preview
Click Rename and Organize

📦 Notes

No installation required
Drag-and-drop works out of the box
If Windows shows a warning, click "More info" → "Run anyway"


⚠️ Disclaimer

Use at your own risk. Always test on a copy of your files first.
